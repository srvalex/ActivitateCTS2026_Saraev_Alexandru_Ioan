public class Computer {
    // Required
    private String cpu;
    private String motherboard;
    private int ramGb;

    // Optional
    private String gpu;           // may be null (integrated graphics)
    private int storageGb;        // default 0
    private boolean hasWifi;      // default false
    private boolean hasBluetooth; // default false

}

**Your task:**

   - Required fields set in the **constructor** of the builder
   - Each optional field as a **chainable setter** returning `this`
   - A `build()` method that returns the final `Computer`

2. Make `Computer`'s constructor **private**, so the only way to create one is through the builder

3. A `toString()` on `Computer` that prints all fields nicely

4. In `Main.java`, build **two** computers:
   - A budget PC (only required fields + storage)
   - A gaming PC (everything set)
