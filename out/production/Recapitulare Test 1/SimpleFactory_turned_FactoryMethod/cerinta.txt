You're building a notification system that can send messages through different channels: Email, SMS, and Push notifications.
The base classes and interfaces are already written for you. Your task is to implement the Factory Method pattern on top of them.


// Notifier.java
public interface Notifier {
    void send(String message);
}

// EmailNotifier.java
public class EmailNotifier implements Notifier {
    public void send(String message) {
        System.out.println("[EMAIL] " + message);
    }
}

// SmsNotifier.java
public class SmsNotifier implements Notifier {
    public void send(String message) {
        System.out.println("[SMS] " + message);
    }
}

// PushNotifier.java
public class PushNotifier implements Notifier {
    public void send(String message) {
        System.out.println("[PUSH] " + message);
    }
}
```

---

**Your task:**

1. An abstract class `NotificationSender` with:
   - An abstract method `createNotifier()` returning a `Notifier`
   - A concrete method `notify(String message)` that calls `createNotifier()` and sends the message

2. Three concrete subclasses: `EmailSender`, `SmsSender`, `PushSender`

3. A `Main.java` with a `main()` method that creates each sender and calls `notify("Hello!")` on each

---

**Expected output:**
```
[EMAIL] Hello!
[SMS] Hello!
[PUSH] Hello!
